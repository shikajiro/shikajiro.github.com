環境構築

bower というのを使えばパッケージ管理できるらしい！すげぇ！（なぜ今まで無かったのか）

bower install d3.js

htmlコードにはこんな感じで

<head>
    ...
    <script src="bower_components/d3/dist/d3.min.js'"></script>
    ...
</head>

D3をとりあえず動かす

適当にpタグでも作っておく

<body>
    <p id="chart"></p>
</body>
